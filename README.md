# Eternal Edge — Stock Market Research

Faith-driven, long-term stock research platform.

## 🚀 Live Site
Deployed automatically via Vercel on every push to `main`.

## 📁 Key File
**The only file you ever need to update:**
```
src/App.jsx
```
This single file contains the entire website. When Claude gives you an updated version, replace this file and push — Vercel deploys automatically in ~60 seconds.

## 🔄 Update Workflow
1. Make changes with Claude
2. Download the new `App.jsx` from Claude
3. Replace `src/App.jsx` in GitHub (Edit → paste → Commit)
4. Vercel auto-deploys in ~60 seconds

## 🛠 Local Development (optional)
```bash
npm install
npm run dev
```
